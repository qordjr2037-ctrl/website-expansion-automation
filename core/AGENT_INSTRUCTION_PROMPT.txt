## Agent가 매 Run 할 일 (멈추지 말 것)

```bash
# 1) supervisor 살아 있는지 (죽었으면 재시작)
bash tools/start_gangara_supervisor.sh

# 2) 상태 확인
cat core/backlink_deploy_status.json | head -30
cat core/backlink_verified_store.json | head -20

# 3) 수집·플랜 (가벼우면 supervisor가 이미 함)
python3 core/run_backlink_collect.py
python3 core/run_backlink_daily_plan.py --target 1000 --quiet

# 4) 배포 시도 (로컬)
python3 core/run_backlink_auto.py --batch 50

# 5) 학습 1 cycle
python3 core/run_learning_loop.py --max-cycles 1
```

**실패 1개**만 고치고 commit·push.  
**사용자가 「그만」** 하기 전까지 supervisor·인텔·auto **중단 금지**.
